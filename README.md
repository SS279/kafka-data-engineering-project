# Introduction
In this project, we have built an End-To-End Data Engineering project on Real-Time Banking data using Kafka.

# Technology used
- Programming Language - Python
- Apache Kafka
- Cloud - Amazon Web Services (AWS)
  1. EC2
  2. S3
  3. Glue Crawler
  4. Glue Catalog
  5. Athena
